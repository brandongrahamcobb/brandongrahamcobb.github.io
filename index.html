OAuthPlugin

A Bukkit/Spigot plugin to link Minecraft accounts with Discord & Patreon via OAuth2.

Getting Started

git clone https://github.com/brandongrahamcobb/oauthplugin.git
    cd oauthplugin
    mvn clean package

Configuration

# plugins/OAuthPlugin/config.yml
    POSTGRES_HOST: "localhost"
    ENABLE_DISCORD: true
    DISCORD_CLIENT_ID: ""
    DISCORD_REDIRECT_URI: "https://your.domain/oauth/discord_callback"
    # …etc…

Example Usage (Java API)

// in your own Bukkit plugin
    Database db = Database.completeGetInstance();
    UserManager um = new UserManager(db);

    CompletableFuture<User> fut = um.getUserByDiscordId(123456789012345678L);
    fut.thenAccept(user -> {
      System.out.println("Patreon Tier: " + user.getPatreonTier());
    });
